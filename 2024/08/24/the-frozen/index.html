为什么能存在？

如果屏幕前的你曾经尝试过网路上各种的“低成本搭建个人博客”之类的 article 的话，那么你有可能对以下指令有些面熟：

[cte] 这篇文章不是教别人怎么他妈用 Hexo 的啊靠！

本地生成，

推送静态网页到远端。

由于个人电脑的效能已经足够渲染这些 HTML 网页，同时极大降低了对云端的需要——你甚至只需要一个静态展示服务就好了——比如 GitHub Pages。

所以这些网站的维护成本——如果不算域名的话——可以视作为 0。

会有什么意义？

任何一句话都有它存在的意义。

这其实指了两件事情。

  • 第一件事：每个人写下的文字都有它的价值，无论是谁，什么话题。

  • 第二件事：你应该回去再看看咱前面有一句很突出的话。

就是这句。SLP_TYPx=S4, SLP_EN

咱之所以用了这一段——是因为咱想表达一种 suspended 的状态。

就如同 Before entering the S4 state, OSPM will save context of all memory as specified in System Address Map Interfaces 一样；

这些特定时间下停止更新的 Blog，可以看作将一个 Time Period 的 Memory 存储到 Blog 这样一个文字的 NVM 当中，以供后人来管中窥豹。

总之…在咱看来，是这样的。

之后又会怎么样？

要么继续被遗忘，

要么想起来翻修，

要么就是被风卷走。

其实也很正常了。博客这个东西，本身其实就已经有些逆时代潮流而行了。

——毕竟没人愿意看很长的文章。也很少有人会愿意写。

所以这种情况其实还算正常的。有些博主会想起来，就会继续写。

其他博主就不一样了。

但这些 S4 的博客给了我们一种新的角度去看待以前的世界。

这也许就是它们存在的意义吧。

其实博主还想写一些的。但想不到了。所以就这样煞尾了。
